Chapter 12 — Cloud Service Orchestration

OCR Pipeline · Vision AI → Pub/Sub → Translation → Pub/Sub → Cloud Storage · 3 Cloud Functions
What is Cloud Service Orchestration?
Orchestration
The coordinated use and execution of components/services for a complex task. Tasks are described as workflow graphs executing processing steps in a loosely coupled manner.
GCP Integration Stack
Infrastructure: Cloud Storage, Dataproc, Dataflow
Integration: Cloud Functions + Pub/Sub
Services: Vision AI, Translation, TTS, etc.
Developer Benefits
✓ No infrastructure management
✓ Focus on application logic only
✓ Auto-scaling handles arbitrary load — zero intervention

Environment Variables

GCP project ID
TRANSLATE_TOPIC
Pub/Sub topic for translation requests (Fn 1 → Fn 2)
TO_LANG
Comma-separated target language codes (e.g. "de,fr,hu")
RESULT_TOPIC
Pub/Sub topic for save requests (Fn 2 → Fn 3)
RESULT_BUCKET
GCS bucket where translated .txt files are saved
OcrTranslateApiMessage Helper

A POJO helper that serialises data between functions via Pub/Sub. Encapsulates the text, filename and target language code for each in-flight message.

Fields:
text — extracted or translated text
filename — original GCS filename
lang — target language code (e.g. "de")
Methods:
toPubsubData() → byte[] for Pub/Sub payload
fromPubsubData(byte[]) → deserialise from Pub/Sub
Full tutorial + source: cloud.google.com/functions/docs/tutorials/ocr
Function 1: OcrProcessImage

Triggered by a Cloud Storage event. Extracts text using Vision AI, detects source language using Translation API, then publishes one Pub/Sub message per target language to TRANSLATE_TOPIC.

Trigger
GCS event (image upload) → accept(GcsEvent, Context)
Services Used
Vision AI (TEXT_DETECTION) + Translation (detectLanguage)
Output
Publishes to TRANSLATE_TOPIC — one message per target language in TO_LANG
public class OcrProcessImage implements BackgroundFunction<GcsEvent> {
    private static final String PROJECT_ID = System.getenv("GCP_PROJECT");
    private static final String TRANSLATE_TOPIC_NAME = System.getenv("TRANSLATE_TOPIC");
    private static final String[] TO_LANGS = System.getenv("TO_LANG").split(",");
    private static final String LOCATION_NAME =
        LocationName.of(PROJECT_ID, "global").toString();
    private Publisher publisher;

    public OcrProcessImage() throws IOException {
        publisher = Publisher.newBuilder(
            ProjectTopicName.of(PROJECT_ID, TRANSLATE_TOPIC_NAME)).build();
    }

    @Override
    public void accept(GcsEvent gcsEvent, Context context) {
        String bucket = gcsEvent.getBucket();
        if (bucket == null) throw new IllegalArgumentException("Missing bucket parameter");
        String filename = gcsEvent.getName();
        if (filename == null) throw new IllegalArgumentException("Missing name parameter");
        detectText(bucket, filename);
    }

    private void detectText(String bucket, String filename) {
        logger.info("Looking for text in image " + filename);

        // Build Vision API request — point to GCS image
        String gcsPath = String.format("gs://%s/%s", bucket, filename);
        ImageSource imgSource = ImageSource.newBuilder().setGcsImageUri(gcsPath).build();
        Image img = Image.newBuilder().setSource(imgSource).build();
        Feature textFeature = Feature.newBuilder()
            .setType(Feature.Type.TEXT_DETECTION).build();
        AnnotateImageRequest visionRequest = AnnotateImageRequest.newBuilder()
            .addFeatures(textFeature).setImage(img).build();
        List<AnnotateImageRequest> visionRequests = new ArrayList<>();
        visionRequests.add(visionRequest);

        // Call Vision AI
        AnnotateImageResponse visionResponse;
        try (ImageAnnotatorClient client = ImageAnnotatorClient.create()) {
            visionResponse = client.batchAnnotateImages(visionRequests).getResponses(0);
            if (visionResponse == null || !visionResponse.hasFullTextAnnotation()) {
                logger.info(String.format("Image %s contains no text", filename));
                return;
            }
            if (visionResponse.hasError()) {
                logger.log(Level.SEVERE, "Vision API error: " +
                    visionResponse.getError().getMessage());
                return;
            }
        } catch (IOException e) {
            logger.log(Level.SEVERE, "Error detecting text: " + e.getMessage(), e);
            return;
        }

        String text = visionResponse.getFullTextAnnotation().getText();
        logger.info("Extracted text: " + text);

        // Detect source language using Translation API
        DetectLanguageRequest languageRequest = DetectLanguageRequest.newBuilder()
            .setParent(LOCATION_NAME)
            .setMimeType("text/plain")
            .setContent(text)
            .build();
        DetectLanguageResponse languageResponse;
        try (TranslationServiceClient client = TranslationServiceClient.create()) {
            languageResponse = client.detectLanguage(languageRequest);
        } catch (IOException e) {
            logger.log(Level.SEVERE, "Error detecting language: " + e.getMessage(), e);
            return;
        }
        if (languageResponse.getLanguagesCount() == 0) {
            logger.info("No languages detected for text: " + text);
            return;
        }
        String languageCode = languageResponse.getLanguages(0).getLanguageCode();
        logger.info(String.format("Detected language %s for %s", languageCode, filename));

        // Publish one translation request per target language
        for (String targetLanguage : TO_LANGS) {
            logger.info("Sending translation request for " + targetLanguage);
            OcrTranslateApiMessage message =
                new OcrTranslateApiMessage(text, filename, targetLanguage);
            ByteString byteStr = ByteString.copyFrom(message.toPubsubData());
            PubsubMessage pubsubApiMessage =
                PubsubMessage.newBuilder().setData(byteStr).build();
            try {
                publisher.publish(pubsubApiMessage).get();
            } catch (InterruptedException | ExecutionException e) {
                logger.log(Level.SEVERE, "Error publishing: " + e.getMessage(), e);
                return;
            }
        }
    }
}
Function 2: OcrTranslateText

Triggered by a Pub/Sub message on TRANSLATE_TOPIC. Deserialises the source text and target language, calls translateText(), then publishes the result to RESULT_TOPIC.

Trigger
Pub/Sub message on TRANSLATE_TOPIC → accept(PubSubMessage, Context)
Services Used
Translation Advanced API (TranslationServiceClient.translateText())
Output
Publishes translated text + filename + lang to RESULT_TOPIC
public class OcrTranslateText implements BackgroundFunction<PubSubMessage> {
    private static final String PROJECT_ID      = getenv("GCP_PROJECT");
    private static final String RESULTS_TOPIC_NAME = getenv("RESULT_TOPIC");
    private static final String LOCATION_NAME   =
        LocationName.of(PROJECT_ID, "global").toString();
    private Publisher publisher;

    public OcrTranslateText() throws IOException {
        publisher = Publisher.newBuilder(
            ProjectTopicName.of(PROJECT_ID, RESULTS_TOPIC_NAME)).build();
    }

    @Override
    public void accept(PubSubMessage pubSubMessage, Context context) {
        // Deserialise the incoming Pub/Sub message
        OcrTranslateApiMessage ocrMessage = OcrTranslateApiMessage.fromPubsubData(
            pubSubMessage.getData().getBytes(StandardCharsets.UTF_8));
        String targetLang = ocrMessage.getLang();
        String text       = ocrMessage.getText();
        logger.info("Translating text into " + targetLang);

        // Build translation request
        TranslateTextRequest request = TranslateTextRequest.newBuilder()
            .setParent(LOCATION_NAME)
            .setMimeType("text/plain")
            .setTargetLanguageCode(targetLang)
            .addContents(text)
            .build();

        // Call Translation API
        TranslateTextResponse response;
        try (TranslationServiceClient client = TranslationServiceClient.create()) {
            response = client.translateText(request);
        } catch (IOException e) {
            logger.log(Level.SEVERE, "Error translating: " + e.getMessage(), e);
            return;
        }
        if (response.getTranslationsCount() == 0) return;

        String translatedText = response.getTranslations(0).getTranslatedText();
        logger.info("Translated text: " + translatedText);

        // Package result and publish to RESULT_TOPIC
        String filename = ocrMessage.getFilename();
        OcrTranslateApiMessage resultMsg =
            new OcrTranslateApiMessage(translatedText, filename, targetLang);
        try {
            ByteString byteStr = ByteString.copyFrom(resultMsg.toPubsubData());
            PubsubMessage pubsubApiMessage =
                PubsubMessage.newBuilder().setData(byteStr).build();
            publisher.publish(pubsubApiMessage).get();
            logger.info("Text translated to " + targetLang);
        } catch (InterruptedException | ExecutionException e) {
            logger.log(Level.SEVERE, "Error publishing result: " + e.getMessage(), e);
        }
    }

    // Guards against ungraceful failures from unset env variables
    private static String getenv(String name) {
        String value = System.getenv(name);
        if (value == null) {
            logger.warning("Environment variable " + name + " was not set");
            value = "MISSING";
        }
        return value;
    }
}
Function 3: OcrSaveResult

Triggered by a Pub/Sub message on RESULT_TOPIC. Constructs the output filename (original_to_lang.txt) and writes the translated text bytes to the RESULT_BUCKET in Cloud Storage.

Trigger
Pub/Sub message on RESULT_TOPIC → accept(PubSubMessage, Context)
Services Used
Cloud Storage (StorageOptions.getDefaultInstance().getService(), STORAGE.create())
Output File Naming
photo.jpg_to_de.txt, photo.jpg_to_fr.txt, etc. — one per target language
public class OcrSaveResult implements BackgroundFunction<PubSubMessage> {
    private static final String RESULT_BUCKET = System.getenv("RESULT_BUCKET");
    // Storage client initialised once in static block (reused across warm invocations)
    private static final Storage STORAGE =
        StorageOptions.getDefaultInstance().getService();

    @Override
    public void accept(PubSubMessage pubSubMessage, Context context) {
        OcrTranslateApiMessage ocrMessage = OcrTranslateApiMessage.fromPubsubData(
            pubSubMessage.getData().getBytes(StandardCharsets.UTF_8));

        logger.info("Received request to save file " + ocrMessage.getFilename());

        // Construct output filename: e.g. "photo.jpg_to_de.txt"
        String newFileName = String.format(
            "%s_to_%s.txt", ocrMessage.getFilename(), ocrMessage.getLang());

        logger.info(String.format("Saving result to %s in bucket %s",
            newFileName, RESULT_BUCKET));

        // Write translated text to GCS
        BlobInfo blobInfo = BlobInfo.newBuilder(
            BlobId.of(RESULT_BUCKET, newFileName)).build();
        STORAGE.create(blobInfo,
            ocrMessage.getText().getBytes(StandardCharsets.UTF_8));

        logger.info("File saved");
    }
}
Deployment & Execution Summary
What to Deploy
1. Deploy all 3 Cloud Functions with env variables set
2. Create input GCS bucket (image source)
3. Create output GCS bucket (RESULT_BUCKET)
4. Create TRANSLATE_TOPIC Pub/Sub topic
5. Create RESULT_TOPIC Pub/Sub topic
How to Test
Upload any image with text to the input GCS bucket.
The pipeline triggers automatically — no other action needed.
Check the output GCS bucket for filename_to_XX.txt files, one per target language.
Key Takeaways
Loose coupling — Functions communicate only via Pub/Sub messages. None of the functions knows about the others directly.
Zero infrastructure — Developer writes only application logic. No servers, no scaling configuration, no ops.
Auto-scaling — Each Cloud Function scales independently. Uploading 1 or 10,000 images is handled transparently.
