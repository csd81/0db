Halmazelmélet

Hartung Ferenc · Pannon Egyetem Matematika Tanszék · 2025  |  Szendrei Ágnes: Diszkrét matematika, Polygon, 2004

Definíció — Halmaz
A halmaz alapfogalom — nem definiáljuk. Intuitívan: jól meghatározott, egymástól megkülönböztethető dolgok összessége.

\(a \in A\) : a eleme az A halmaznak — szintén alapfogalom.
Példák
\(A = \{1, 5, 7\}\)  →  \(5 \in A,\quad 4 \notin A\)
\(B = \{x \in \mathbb{R} : x \ge 3 \text{ és } x < 10\}\)  → \(3 \in B,\; 10 \notin B\)
\(C = \{1, A, \{4\}\}\)  →  \(4 \notin C,\quad \{4\} \in C\)
\(D = \{2, 4, 6, 8, \ldots\}\)  (páros pozitív egészek)
Egyenlőség & Üres halmaz
Az A és B halmazok egyenlőek, ha ugyanazok az elemeik:
\[\{1,5,8\} = \{1,8,5\} = \{1,8,1,5\} \;\neq\; \big\{\{1\},\{5\},8\big\}\] Azt a halmazt, amelynek nincs eleme, üres halmaznak hívjuk: \(\emptyset\).
\(U\) — az univerzális halmaz (az összes vizsgált elem halmaza).
Számhalmazok jelölései
\(\mathbb{R}\) valós számok halmaza
\(\mathbb{Q}\) racionális számok — két egész szám hányadosaként felírható valós számok
\(\mathbb{Z}\) egész számok halmaza
\(\mathbb{N}\) pozitív egész számok: \(\mathbb{N} = \{1, 2, 3, 4, \ldots\}\)
\(\mathbb{N}_0\) nemnegatív egész számok: \(\mathbb{N}_0 = \{0, 1, 2, 3, \ldots\}\)
Összefüggés: \(\mathbb{N} \subset \mathbb{N}_0 \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}\)

Definíció — Részhalmaz
Azt mondjuk, hogy A részhalmaza B-nek, ha A minden eleme eleme B-nek is.
Jelölés: \(A \subseteq B\)  (vagy \(B \supseteq A\))
Definíció — Valódi részhalmaz
A valódi részhalmaza B-nek, ha \(A \subseteq B\) és \(A \neq B\).
Jelölés: \(A \subset B\)
⚠ Néhány könyv az \(A \subset B\) jelölést a részhalmaz (nem valódi) jelölésére is használja.
Példa
\(\{1,3,5\} \subseteq \{1,2,3,4,5\}\),  sőt  \(\{1,3,5\} \subset \{1,2,3,4,5\}\)
De  \(\{1,2,3,4,5\} \not\subseteq \{1,3,5\}\)
Tétel — Részhalmaz tulajdonságai
Tetszőleges \(A, B, C\) halmazokra:
  1. \(\emptyset \subseteq A\)  és  \(A \subseteq A\)
  2. ha \(A \subseteq B\) és \(B \subseteq C\), akkor \(A \subseteq C\)  (tranzitivitás)
  3. \(A = B \;\Longleftrightarrow\; A \subseteq B \text{ és } B \subseteq A\)  (antiszimmetria)
Definíció — Diszjunkt halmazok
Az A és B halmazok diszjunktak, ha \(A \cap B = \emptyset\).
\(A \subseteq B\) — Venn-diagram
Ha \(A \subseteq B\), akkor \(A\) belül van \(B\)-ben
Definíció — Hatványhalmaz
Az A halmaz összes részhalmazainak halmazát A hatványhalmazának hívjuk.
Jelölés: \(\mathcal{P}(A)\)
Példa — Hatványhalmaz
\(A = \{a, b, c\}\)
\(\mathcal{P}(A) = \bigl\{\,\emptyset,\,\{a\},\,\{b\},\,\{c\},\,\{a,b\},\,\{a,c\},\,\{b,c\},\,\{a,b,c\}\,\bigr\}\)
Ha \(|A| = n\), akkor \(|\mathcal{P}(A)| = 2^n\).
Jelen esetben: \(2^3 = 8\) elem ✓

Összefoglalás
\(A \cup B\) — Unió
\(\{x \in U : x \in A \;\text{vagy}\; x \in B\}\)
\(A \cap B\) — Metszet
\(\{x \in U : x \in A \;\text{és}\; x \in B\}\)
\(A \setminus B\) — Különbség
\(\{x \in U : x \in A \;\text{és}\; x \notin B\}\)
\(A \mathbin{\triangle} B\) — Szimmetrikus különbség

\(= (A \setminus B) \cup (B \setminus A) = (A \cap \overline{B}) \cup (\overline{A} \cap B)\)
\(\overline{A}\) — Komplemens
\(\{x \in U : x \notin A\}\)

De Morgan azonosságok
\[\overline{A \cap B} = \overline{A} \cup \overline{B}\] \[\overline{A \cup B} = \overline{A} \cap \overline{B}\] A metszet komplementere = komplementek uniója, és fordítva.
Tétel — Különbség és szimm. különbség
\[A \setminus B = A \cap \overline{B}\] \[A \mathbin{\triangle} B = (A \cup B) \setminus (A \cap B) = (A \setminus B) \cup (B \setminus A) = (A \cap \overline{B}) \cup (\overline{A} \cap B)\]
Alaptulajdonságok
\(\overline{\overline{A}} = A\)
\(A \cap \overline{A} = \emptyset\)
\(A \cup \overline{A} = U\)
\(A \cap U = A\)
\(A \cup U = U\)
\(A \cap \emptyset = \emptyset\)
\(A \cup \emptyset = A\)
Részhalmaz-jellemzések (\(A \subseteq B\) egyenértékű feltételei)
\(A \subseteq B \;\Longleftrightarrow\; A \cup B = B\)
\(A \subseteq B \;\Longleftrightarrow\; A \cap B = A\)
\(A \subseteq B \;\Longleftrightarrow\; A \cap \overline{B} = \emptyset\)
Indexelt halmazok
Legyen \(I\) indexhalmaz, \(A_i \subseteq U\) minden \(i \in I\)-re. Ekkor: \[\bigcap_{i \in I} A_i = \{x \in U : x \in A_i \text{ minden } i \in I\text{-re}\}\] \[\bigcup_{i \in I} A_i = \{x \in U : x \in A_i \text{ valamely } i \in I\text{-re}\}\]
1. Bizonyítás — Venn-diagrammal
Mutassuk meg: \((A \cap B) \cup C = (A \cup C) \cap (B \cup C)\)
Lépésről lépésre Venn-diagrammal (az előadás 15–18. diái):
1.  Jelöljük be \(A \cap B\)-t (a bal körök metszete)
2.  Az \((A \cap B) \cup C\) területe = bal oldal
3.  Jelöljük be \(A \cup C\)-t (A és C körök tele)
4.  Jelöljük be \(B \cup C\)-t (B és C körök tele)
5.  \((A \cup C) \cap (B \cup C)\) metszete = azonos terület a bal oldallal \(\blacksquare\)
2. Bizonyítás — Algebrai
Mutassuk meg: \((A \cup B) \setminus C = (A \setminus C) \cup (B \setminus C)\)
\((A \cup B) \setminus C\)
\(= (A \cup B) \cap \overline{C}\)
\(A \setminus X = A \cap \overline{X}\)
\(= (A \cap \overline{C}) \cup (B \cap \overline{C})\)
disztributivitás
\(= (A \setminus C) \cup (B \setminus C) \quad\blacksquare\)
\(A \cap \overline{X} = A \setminus X\) visszaalakítva
3. Bizonyítás — Algebrai (hosszabb)
Mutassuk meg: \(A \mathbin{\triangle} (A \mathbin{\triangle} B) = B\)
Felhasználjuk: \(X \mathbin{\triangle} Y = (X \cap \overline{Y}) \cup (\overline{X} \cap Y)\)
Legyen \(X = A \mathbin{\triangle} B\)
rövidítés
\(X = (A \cap \overline{B}) \cup (\overline{A} \cap B)\)
\(\triangle\) definíciója
\(\overline{X} = (\overline{A} \cup B) \cap (A \cup \overline{B})\)
de Morgan kétszer
\(A \cap \overline{X} = (\overline{A} \cap B \cap A) \cup (A \cap A \cap \overline{B})\)
kifejtés disztributivitással
\(\phantom{A \cap \overline{X}\,}= \emptyset \cup (A \cap \overline{B}) = A \cap \overline{B}\)
\(A \cap \overline{A} = \emptyset\)
\(\overline{A} \cap X = (\overline{A} \cap A \cap \overline{B}) \cup (\overline{A} \cap \overline{A} \cap B)\)
kifejtés
\(\phantom{\overline{A} \cap X\,}= \emptyset \cup (\overline{A} \cap B) = \overline{A} \cap B\)
\(\overline{A} \cap A = \emptyset\)
\(A \mathbin{\triangle} X = (A \cap \overline{B}) \cup (\overline{A} \cap B)\)
összegyűjtve
\(\phantom{A \mathbin{\triangle} X\,}= (A \cup \overline{A}) \cap B\)
disztributivitás
\(\phantom{A \mathbin{\triangle} X\,}= U \cap B\)
\(A \cup \overline{A} = U\)
\(= B \quad\blacksquare\)
\(U \cap B = B\)
Következmény: Ha \(C = A \mathbin{\triangle} B\), akkor \(A \mathbin{\triangle} C = B\) — azaz \(\triangle\) "visszacsinálható". Minden elem saját inverze a \((\mathcal{P}(U),\, \triangle,\, \emptyset)\) Abel-csoportban.
