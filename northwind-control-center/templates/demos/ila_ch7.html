ILA — 7. fejezet

Komplex számok

Hartung Ferenc · Pannon Egyetem Matematika Tanszék · 2025

Komplex számok bevezetése és alapműveletek
Az \(a + bi\) alakú kifejezéseket komplex számoknak nevezzük, ahol \(a, b \in \mathbb{R}\) és \(i^2 = -1\).
\(\mathbb{C} := \{a + bi : a, b \in \mathbb{R}\}\)
Ha \(z = x + yi\):   \(\operatorname{Re} z = x\) (valós rész),   \(\operatorname{Im} z = y\) (képzetes rész).
Összeadás / Kivonás:
\(z_1 \pm z_2 = (x_1 \pm x_2) + (y_1 \pm y_2)i\)
Szorzás:
\(z_1 z_2 = (x_1 x_2 - y_1 y_2) + (x_1 y_2 + x_2 y_1)i\)
Osztás:
\(\dfrac{z_1}{z_2} = \dfrac{x_1 x_2 + y_1 y_2}{x_2^2 + y_2^2} + \dfrac{x_2 y_1 - x_1 y_2}{x_2^2 + y_2^2}\,i\)

Interaktív komplex kalkulátor:

\(z_1\): + i
\(z_2\): + i
Komplex konjugált és abszolút érték
Konjugált: \(\bar{z} = x - yi\)
Geometriailag: tükrözés a valós tengelyre.
Abszolút érték: \(|z| = \sqrt{x^2 + y^2}\)
Az origótól mért távolság a komplex síkon.
Főbb tulajdonságok: \(\bar{\bar{z}} = z;\quad \overline{z_1 z_2} = \bar{z}_1 \bar{z}_2;\quad z + \bar{z} = 2\operatorname{Re}z;\quad z\bar{z} = |z|^2\)
\(|z_1 z_2| = |z_1||z_2|;\quad \bigl||z_1| - |z_2|\bigr| \le |z_1 + z_2| \le |z_1| + |z_2|\)  (háromszög-egyenlőtlenség)

Húzd a pontot a komplex síkon:

Trigonometrikus alak & De Moivre-formula
Legyen \(z = x + yi \ne 0\). A trigonometrikus alak:
\[z = r(\cos\varphi + i\sin\varphi),\quad r = |z| = \sqrt{x^2+y^2},\quad \varphi = \arg z\] ahol \(x = r\cos\varphi\) és \(y = r\sin\varphi\).
Szorzás és osztás trigonometrikus alakban:
\(z_1 z_2 = r_1 r_2\bigl(\cos(\varphi+\psi) + i\sin(\varphi+\psi)\bigr)\)
\(\dfrac{z_1}{z_2} = \dfrac{r_1}{r_2}\bigl(\cos(\varphi-\psi) + i\sin(\varphi-\psi)\bigr)\)
De Moivre-formula: \(z^n = r^n\bigl(\cos(n\varphi) + i\sin(n\varphi)\bigr)\) minden \(n \in \mathbb{Z}\)-re.

De Moivre példa (előadásból):
\((-1+i)^{50}\): mivel \(-1+i = \sqrt{2}\bigl(\cos\tfrac{3\pi}{4}+i\sin\tfrac{3\pi}{4}\bigr)\),
\((-1+i)^{50} = (\sqrt{2})^{50}\bigl(\cos\tfrac{150\pi}{4}+i\sin\tfrac{150\pi}{4}\bigr) = 2^{25}\bigl(\cos\tfrac{3\pi}{2}+i\sin\tfrac{3\pi}{2}\bigr) = -2^{25}i\)
Komplex n-edik gyökök
Legyen \(z = r(\cos\varphi + i\sin\varphi)\), \(n \in \mathbb{N}\). Ekkor az összes n-edik gyök: \[\sqrt[n]{z} = \sqrt[n]{r}\left(\cos\frac{\varphi+2k\pi}{n} + i\sin\frac{\varphi+2k\pi}{n}\right), \quad k = 0,1,\ldots,n-1.\] A gyökök szabályos n-szöget alkotnak a komplex síkon, középpontban az origóval, sugaruk \(\sqrt[n]{r}\).
Ha \(w_0\) az első n-edik gyök, akkor az összes gyök: \(w_0,\; w_0\varepsilon_1,\; w_0\varepsilon_1^2,\;\ldots,\; w_0\varepsilon_1^{n-1}\), ahol \(\varepsilon_1 = e^{2\pi i/n}\).

Komplex egységgyökök & Euler-formula
Az 1 komplex szám n-edik gyökeit n-edik egységgyököknek nevezzük:
\[\varepsilon_k := \cos\frac{2k\pi}{n} + i\sin\frac{2k\pi}{n}, \quad k=0,1,\ldots,n-1.\] Az egységgyökök halmaza: \(\{1,\varepsilon_1,\varepsilon_1^2,\ldots,\varepsilon_1^{n-1}\}\).
Az n-edik egységgyökök összege: \(\displaystyle\sum_{k=0}^{n-1}\varepsilon_k = \frac{1-\varepsilon_1^n}{1-\varepsilon_1} = 0\)
Euler-formula: \(e^{i\varphi} = \cos\varphi + i\sin\varphi\)
Exponenciális alak: \(z = re^{i\varphi}\)
Következmény: \(\cos\varphi = \dfrac{e^{i\varphi}+e^{-i\varphi}}{2},\quad \sin\varphi = \dfrac{e^{i\varphi}-e^{-i\varphi}}{2i}\)

n-edik egységgyökök vizualizálója:

cos 5x és sin 5x kifejtése (előadásból):
Mivel \(z^5 = (\cos x + i\sin x)^5\) és \(z^5 = \cos 5x + i\sin 5x\) (De Moivre):
\(\cos 5x = \cos^5 x - 10\cos^3 x\sin^2 x + 5\cos x\sin^4 x\)
\(\sin 5x = 5\cos^4 x\sin x - 10\cos^2 x\sin^3 x + \sin^5 x\)
Előadás+ — Kiegészítő magyarázatok
1. Történeti háttér — miért kellettek a komplex számok?
A valós számok körén belül az \(x^2 = -1\) egyenletnek nincs megoldása — nincs olyan valós szám, amelynek négyzete negatív.

Problémák: a másodfokú egylet megoldóképlete gyak. negatív szám alatti gyököt ad.
Bolyai János (és más mathematikusok) merész lépése: legyen i jelölés arra a szimbólumra, amelyre i²=−1.
„Képzetes" → formális
Sokáig „képzetes számoknak" (imaginary numbers) hívták, mert nem volt valódi geometriai értelmük. A Gauss-sík (komplex sík) megmutatta: \(a + bi\) = pont az \((a, b)\) síkban. Ettől lett konkrét.
2. Komplex számok mint rendezett párok
Formálisan: \(\mathbb{C} := \mathbb{R}^2\) mint halmaz, ahol az összeadás = vektorok összeadása:
\((a,b) + (c,d) = (a+c,\; b+d)\)

A szorzást külön definiáljuk:
\((a,b) \cdot (c,d) = (ac - bd,\; ad + bc)\)

\(\mathbf{i} := (0, 1)\), ekkor \(\mathbf{i}^2 = (0,1)\cdot(0,1) = (-1, 0) = -1\). ✓
Valós számok: \(a \leftrightarrow (a, 0)\). Így \(\mathbb{R} \subset \mathbb{C}\).
Az előadásból: az összeadás ugyanaz, mint számpárokra, vektorokra — a geometria és az algebra összekapcsolódik.
\((-2+5i) + (4+3i) = 2 + 8i\)  ←  valós és képzetes részeket külön adjuk össze.
3. Az i hatványainak 4-es ciklusa
\(i^1\) \(i^2\) \(i^3\) \(i^4\) \(i^5\) \(i^n\)
\(i\) \(-1\) \(-i\) \(1\) \(i\) \(i^{n \bmod 4}\)
Pl. \(i^{37} = i^{36+1} = (i^4)^9 \cdot i = 1^9 \cdot i = i\). Elég \(n \bmod 4\)-et nézni.
4. ℂ mezőtulajdonságai
Az előadásból: az összeadás és szorzás kommutatív, asszociatív, a szorzás disztributív az összeadásra nézve. Tehát \((\mathbb{C}, +, \cdot)\) test (mező / field).
\(z_1 + z_2 = z_2 + z_1\)  ·  kommutatív összeadás
\(z_1 \cdot z_2 = z_2 \cdot z_1\)  ·  kommutatív szorzás
\(z(z_1 + z_2) = zz_1 + zz_2\)  ·  disztributivitás
Minden \(z \ne 0\) komplex számnak van inverze: \(z^{-1} = \bar{z}/|z|^2\). Ezért osztani is lehet (nullával kivéve).
5. Osztás — a konjugált szerepe
Cél: \(\dfrac{z_1}{z_2}\) kifejezése \(a+bi\) alakban. Módszer: bővítsd a számláló + nevező mindkettőjét \(\bar{z}_2\)-vel. \[\frac{z_1}{z_2} = \frac{z_1 \cdot \bar{z}_2}{z_2 \cdot \bar{z}_2} = \frac{z_1 \cdot \bar{z}_2}{|z_2|^2}\] A nevező \(z_2 \cdot \bar{z}_2 = |z_2|^2\) — valós szám! Csak egyszerű valós osztás marad.
Előadás-példa: \(\dfrac{-2+5i}{4+3i}\)
Bővítés: \(\dfrac{(-2+5i)(4-3i)}{(4+3i)(4-3i)}\)
Nevező: \(4^2+3^2 = 25\)
Számláló: \((-2)(4)+(-2)(-3i)+(5i)(4)+(5i)(-3i)\)
\(= -8 + 6i + 20i - 15i^2 = -8 + 26i + 15 = 7 + 26i\)
Eredmény: \(\dfrac{7+26i}{25} = \dfrac{7}{25} + \dfrac{26}{25}i\)
6. Argumentum meghatározás — először rajzolj!
Az előadó tanácsa: „Mindig azt javaslom, hogy először rajzoljuk le a komplex számunkat a komplex síkon."

Módszer a φ = arg(z) meghatározáshoz:
  1. Ábrázold a \((x, y)\) pontot — melyik negyedben van?
  2. Számítsd \(r = \sqrt{x^2+y^2}\)-t.
  3. Speciális szögnél (30°, 45°, 60°) olvasd le az ábrából — nem kell arctan.
  4. Általános esetben: \(\varphi = \arctan(y/x)\), de figyelj a negyedre!
-1+i: Re=-1, Im=1 → 2. negyedben, \(r=\sqrt{2}\).
A derékszögű háromszög befogói egyenlők (|Re|=|Im|=1) → 45°-os szög a tengelytől.
\(\varphi = \tfrac{\pi}{2} + \tfrac{\pi}{4} = \tfrac{3\pi}{4}\) — leolvasható az ábrából, arctan sem kell.
Trig. alak: \(-1+i = \sqrt{2}\!\left(\cos\tfrac{3\pi}{4} + i\sin\tfrac{3\pi}{4}\right)\)
7. Összetett trig. alak — szorzás/osztás egyszerre
Szorzásnál: modulusokat megszorozzuk, szögeket összeadjuk.
Osztásnál: modulusokat elosztjuk, szögeket kivonjuk.
Hatvány: \(z^n = r^n(\cos n\varphi + i\sin n\varphi)\)
Konjugált trig. alakban: \(\bar{z} = r(\cos\varphi - i\sin\varphi) = r(\cos(-\varphi) + i\sin(-\varphi))\)
Előadás-feladat: Legyen \(z_1 = \cos\tfrac{\pi}{3}+i\sin\tfrac{\pi}{3}\) (r=1, φ=π/3),   \(z_2 = 2(\cos\tfrac{\pi}{4}+i\sin\tfrac{\pi}{4})\),   \(z_3 = 2(\cos\tfrac{\pi}{6}+i\sin\tfrac{\pi}{6})\).
Számítsuk: \(\dfrac{z_1^4 \cdot \bar{z}_2}{z_3}\)
\(z_1^4\):   \(r=1^4=1\),   \(\varphi=4\cdot\tfrac{\pi}{3}=\tfrac{4\pi}{3}\)
\(\bar{z}_2\):   \(r=2\),   \(\varphi=-\tfrac{\pi}{4}\)
\(z_1^4\cdot\bar{z}_2\):   \(r=1\cdot2=2\),   \(\varphi=\tfrac{4\pi}{3}-\tfrac{\pi}{4}=\tfrac{13\pi}{12}\)
osztás \(z_3\)-mal:   \(r=2/2=1\),   \(\varphi=\tfrac{13\pi}{12}-\tfrac{\pi}{6}=\tfrac{11\pi}{12}\)
Eredmény: \(\cos\tfrac{11\pi}{12} + i\sin\tfrac{11\pi}{12}\)
