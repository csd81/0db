High-Level GCP AI/ML Services

Beyond IaaS and batch/stream compute, GCP offers ready-to-use service components that dramatically increase application capabilities while reducing development effort. The most exciting are in AI and human-machine interfacing.

Service Capabilities Chapter

Vision AI Image detection and recognition without training: text extraction (OCR), image labelling, face and landmark detection, content tagging, object localisation, product search, safe-search detection. § 11.1 ✓

Cloud Natural Language Sentiment analysis, entity analysis, entity sentiment analysis, content classification, and syntax analysis from unstructured text. Overview

Vision AI — ImageAnnotatorClient
Client Usage Pattern
1. Create ImageAnnotatorClient (try-with-resources)
2. Build a List<AnnotateImageRequest> with features
3. Call batchAnnotateImages(requests)
4. Process List<AnnotateImageResponse>
5. Client auto-closed by try-with-resources
Works on Local & Remote Images
The img parameter in AnnotateImageRequest.setImage(img) can reference a local file or a remote URL (GCS URI or public HTTPS). The API handles the transport transparently.
Client Creation & Response Processing
// Client is created once and can be reused for multiple requests.
// try-with-resources ensures close() is called to free background resources.
try (ImageAnnotatorClient client = ImageAnnotatorClient.create()) {

    BatchAnnotateImagesResponse response = client.batchAnnotateImages(requests);
    List<AnnotateImageResponse> responses = response.getResponsesList();

    for (AnnotateImageResponse res : responses) {
        if (res.hasError()) {
            System.out.printf("Error: %s\n", res.getError().getMessage());
            return;
        }
        // process individual detection results ...
    }
}
Building a Request — Single Feature (TEXT_DETECTION)
List<AnnotateImageRequest> requests = new ArrayList<>();

Feature feat = Feature.newBuilder()
    .setType(Feature.Type.TEXT_DETECTION)
    .build();

AnnotateImageRequest request = AnnotateImageRequest.newBuilder()
    .addFeatures(feat)
    .setImage(img)   // local or remote image
    .build();

requests.add(request);
Multi-Feature Request — Text + Document Text + Logos
List<AnnotateImageRequest> requests = new ArrayList<>();

// Feature 1: printed text / OCR
Feature feat1 = Feature.newBuilder().setType(Feature.Type.TEXT_DETECTION).build();
AnnotateImageRequest req1 = AnnotateImageRequest.newBuilder()
    .addFeatures(feat1).setImage(img).build();

// Feature 2: handwritten text (dense document OCR)
Feature feat2 = Feature.newBuilder()
    .setType(Feature.Type.DOCUMENT_TEXT_DETECTION).build();
AnnotateImageRequest req2 = AnnotateImageRequest.newBuilder()
    .addFeatures(feat2).setImage(img).build();

// Feature 3: corporate logos
Feature feat3 = Feature.newBuilder().setType(Feature.Type.LOGO_DETECTION).build();
AnnotateImageRequest req3 = AnnotateImageRequest.newBuilder()
    .addFeatures(feat3).setImage(img).build();

requests.add(req1);
requests.add(req2);
requests.add(req3);

Cloud Translation Java API
Entry Point
Package: com.google.cloud.translate

Interface Translate is a service proxy. Obtained via TranslateOptions.getDefaultInstance().getService().
Translation Models
base — Standard translation model. Fast, lower cost.
nmt — Neural Machine Translation. Premium, higher quality, more natural results.
Advanced API
TranslationServiceClient + client.translateText(request) adds: translation glossaries, batch translation for large text volumes, custom models.
Auto-Detection (Simplest Case)
import com.google.cloud.translate.*;

// Obtain default Translate service (uses application default credentials)
Translate translate = TranslateOptions.getDefaultInstance().getService();

// Language auto-detected from source text
Translation translation = translate.translate("¡Hola Mundo!");

System.out.printf("Translated Text:\n\t%s\n", translation.getTranslatedText());
// → "Hello World!"
Explicit Source / Target Languages + NMT Model
Translation translation = translate.translate(
    "Hola Mundo!",
    Translate.TranslateOption.sourceLanguage("es"),   // Spanish source
    Translate.TranslateOption.targetLanguage("de"),   // German target
    // "base" for standard edition, "nmt" for premium neural MT
    Translate.TranslateOption.model("nmt")
);
// → "Hallo Welt!"

Text-to-Speech — QuickstartSample
Key Classes
TextToSpeechClient — main service client
SynthesisInput — text or SSML source
VoiceSelectionParams — language + gender
AudioConfig — output format (MP3, WAV, OGG)
SynthesizeSpeechResponse — audio bytes
SsmlVoiceGender — NEUTRAL / MALE / FEMALE
AudioEncoding — MP3 / LINEAR16 / OGG_OPUS
Synthesis Steps
1. Create TextToSpeechClient (try-with-resources)
2. Build SynthesisInput with text or SSML
3. Set VoiceSelectionParams: language code + gender
4. Set AudioConfig: output encoding (MP3)
5. Call synthesizeSpeech(input, voice, audioConfig)
6. Extract ByteString audioContents from response
7. Write bytes to output file (output.mp3)
Java — QuickstartSample (Full)
import com.google.cloud.texttospeech.v1.AudioConfig;
import com.google.cloud.texttospeech.v1.AudioEncoding;
import com.google.cloud.texttospeech.v1.SsmlVoiceGender;
import com.google.cloud.texttospeech.v1.SynthesisInput;
import com.google.cloud.texttospeech.v1.SynthesizeSpeechResponse;
import com.google.cloud.texttospeech.v1.TextToSpeechClient;
import com.google.cloud.texttospeech.v1.VoiceSelectionParams;
import com.google.protobuf.ByteString;
import java.io.FileOutputStream;
import java.io.OutputStream;

public class QuickstartSample {
    public static void main(String... args) throws Exception {

        try (TextToSpeechClient textToSpeechClient = TextToSpeechClient.create()) {

            // 1. Text to synthesise
            SynthesisInput input = SynthesisInput.newBuilder()
                .setText("Hello, World!")
                .build();

            // 2. Voice: language code + gender
            VoiceSelectionParams voice = VoiceSelectionParams.newBuilder()
                .setLanguageCode("en-US")
                .setSsmlGender(SsmlVoiceGender.NEUTRAL)
                .build();

            // 3. Output format: MP3
            AudioConfig audioConfig = AudioConfig.newBuilder()
                .setAudioEncoding(AudioEncoding.MP3)
                .build();

            // 4. Perform TTS request
            SynthesizeSpeechResponse response =
                textToSpeechClient.synthesizeSpeech(input, voice, audioConfig);

            // 5. Extract audio bytes from response
            ByteString audioContents = response.getAudioContent();

            // 6. Write to file
            try (OutputStream out = new FileOutputStream("output.mp3")) {
                out.write(audioContents.toByteArray());
                System.out.println("Audio content written to file \"output.mp3\"");
            }
        }
    }
}
Voice Options
Language codes: en-US, de-DE, hu-HU, ja-JP …
Genders: NEUTRAL / MALE / FEMALE
Full list: cloud.google.com/text-to-speech/docs/voices
SSML Support
Instead of .setText(), use .setSsml() to pass Speech Synthesis Markup Language for fine-grained control of pronunciation, pauses, emphasis, and pitch.
