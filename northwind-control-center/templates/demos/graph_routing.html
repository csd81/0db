The EuropeGraph dataset encodes real road and ferry distances between major European cities as a weighted directed graph where edge weights are kilometres. Dijkstra's algorithm finds the shortest path by greedily expanding the lowest-cost frontier node; A* improves on Dijkstra by using the geodesic straight-line distance to the destination as an admissible heuristic, pruning branches that cannot possibly beat the current best path and reducing explored nodes by 60–80%. Configuring the ferry penalty multiplier shows how pricing asymmetry between road and maritime routes changes the optimal itinerary, exactly as it does in real logistics routing systems. This is the algorithm inside every navigation app, package router, and network packet scheduler on the planet.
