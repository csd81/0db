Load Northwind data from a registered connection into the in-memory SQLite store. Once loaded, aggregation queries run instantly without hitting the source database.
How It Works
  1. Select a registered SQL Server or SQLite connection as the source.
  2. Click Load Northwind — the app fetches Orders, Order Details, Products, Categories, and Customers via adapter_select().
  3. Pandas joins and flattens the data into a fact_sales table in an in-memory sqlite3.connect(':memory:') database.
  4. Aggregation queries run against the in-memory store — no network round-trips.

Memory usage: ~10–30 MB for the full Northwind dataset.
